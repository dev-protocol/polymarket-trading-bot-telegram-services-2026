- 🤖 **Интеграция с Synth AI edge** — Запрос вероятностных прогнозов Bittensor SN50 (Synth) через SDK/API → обнаружение и авто-торговля мисценообразованиями 5-15%+ по сравнению с имплицитными шансами Polymarket на рынках BTC/ETH/SOL (почасовые/15-мин/дневные Up-Down и range-рынки) — топ-стратегия 2026 года  
- 👥 **Поддержка нескольких трейдеров** — копирование/следование сразу нескольким успешным трейдерам для диверсифицированной альфы  
- 📏 **Динамический размер ордера** — PERCENTAGE (например 10% от размера трейдера), FIXED (фиксированная сумма в USD), ADAPTIVE (масштабирование по капиталу)  
- 🔝 **Многоуровневые множители** — разные правила масштабирования для крупных и мелких копируемых сделок  
- 📊 **Точная бухгалтерия позиций** — отслеживание каждой покупки/продажи для сохранения корректной чистой позиции даже при частичном исполнении или изменениях баланса  
- 📦 **Пакетная обработка и агрегация ордеров** — объединение мелких сигналов в fewer газоэффективные крупные ордера  
- ⚡ **Субсекундный / низколатентный мониторинг и исполнение** — размещение ордеров в реальном времени в CLOB Polymarket  
- 🛡️ **Защита от проскальзывания** — отклонение исполнений хуже допустимых ценовых порогов  
- 💰 **Отслеживание и мониторинг PnL** — расчёт прибыли/убытка в реальном времени (добавлено в феврале 2026)  
- 🏥 **Проверки здоровья и живой статус** — через Telegram (балансы, логи, состояние бота)  
- 🔄 **Управление старт/стоп и конфигурацией** — полный контроль из Telegram без локального терминала  
- 💾 **Постоянное состояние на MongoDB** — хранение сделок, позиций, истории для воспроизведения, анализа и восстановления  
- 🔍 **Детекция арбитража по задержке** — эксплуатация лагов цены (биржа vs. котировки Polymarket) — отдельный бот + примеры  
- ⚡ **Снайпинг конца цикла / 5 минут** — быстрое обнаружение движений крипты → вход до корректировки на рынках «Up or Down»  
- 📈 **Поддержка маркет-мейкинга** — управление инвентарём, размещение котировок, cancel/replace, контроль рисков (отдельный модуль)  
- 🧪 **Режим предпросмотра / симуляции** — `PREVIEW_MODE=true` отслеживает сигналы без реального исполнения (безопасное тестирование)  
- 🔐 **Контроль рисков** — максимальный/минимальный размер ордера (`MAX_ORDER_SIZE_USD`, `MIN_ORDER_SIZE_USD`), лимиты на сделку/день в примерах арбитража  
- 🌐 **Поддержка мульти-рынков и хеджирования** — хеджирование Yes/No, обработка выкупа, совместимость между рынками  
- 📡 **Интерфейс и интеграция Telegram** — настройка, мониторинг, логи, операции с кошельком, корректировка стратегий
