- 👥 **Поддержка нескольких трейдеров** — копирование/следование сразу нескольким успешным трейдерам для диверсифицированного alpha  
- 📏 **Динамический размер ордера** — PERCENTAGE (например 10% от размера трейдера), FIXED (фиксированная сумма в USD), ADAPTIVE (масштабирование по капиталу)  
- 🔝 **Многоуровневые множители** — разные коэффициенты для крупных и мелких копируемых сделок  
- 📊 **Точная бухгалтерия позиций** — учёт каждого buy/sell для сохранения корректной чистой позиции даже при частичном исполнении  
- 📦 **Пакетная обработка и агрегация ордеров** — объединение мелких сигналов в крупные газоэффективные ордера  
- ⚡ **Субсекундный мониторинг и исполнение с низкой задержкой** — размещение ордеров в реальном времени в CLOB Polymarket  
- 🛡️ **Защита от проскальзывания** — отклонение исполнения по цене хуже допустимой  
- 💰 **Отслеживание и мониторинг PnL** — расчёт прибыли/убытка в реальном времени (добавлено в феврале 2026)  
- 🏥 **Проверка состояния и живой статус** — через Telegram (балансы, логи, состояние бота)  
- 🔄 **Управление старт/стоп и конфигурацией** — полный контроль из Telegram без локального терминала  
- 💾 **Постоянное состояние на MongoDB** — хранение сделок, позиций, истории для воспроизведения, анализа и восстановления  
- 🔍 **Детекция арбитража задержки** — использование разницы цен (биржа vs. котировки Polymarket) — отдельный бот + примеры  
- ⚡ **Снайпинг конца цикла / 5 минут** — обнаружение быстрых движений крипты → вход до корректировки на рынках Up/Down  
- 📈 **Поддержка маркет-мейкинга** — управление инвентарём, размещение котировок, cancel/replace, контроль рисков (отдельный модуль)  
- 🧪 **Режим предпросмотра / симуляции** — `PREVIEW_MODE=true` отслеживает сигналы без исполнения (безопасное тестирование)  
- 🔐 **Контроль рисков** — максимальный/минимальный размер ордера (`MAX_ORDER_SIZE_USD`, `MIN_ORDER_SIZE_USD`), лимиты на сделку/день в примерах арбитража  
- 🌐 **Поддержка мульти-рынков и хеджирования** — хеджирование Yes/No, обработка выкупа, совместимость между рынками  
- 📡 **Интерфейс и интеграция Telegram** — настройка, мониторинг, логи, операции с кошельком, корректировка стратегий  
- ⏱️ **Оптимизация под VPS с низкой задержкой** — <1 мс до нод Polygon, обход GEO-ограничений (рекомендуется AMS)
